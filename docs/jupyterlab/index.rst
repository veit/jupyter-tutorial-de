JupyterLab
==========

`JupyterLab <https://jupyterlab.readthedocs.io/en/latest/index.html>`_ ist ein
erweiterbarer, funktionsreicher Editor zum Erstellen und Bearbeiten von
:doc:`Jupyter Notebooks <../notebook/index>`:

* Ihr könnt mehrere Dokumente und Aktivitäten in eurem Arbeitsbereich mit Hilfe
  von Registerkarten nebeneinander anordnen.
* Code-Konsolen bieten temporäre Scratchpads für die interaktive Ausführung von
  Code, die auch mit einem :doc:`Notebook-Kernel <../kernels/index>` verknüpft
  werden können.
* Es gibt auch eine Vorschau von
  :doc:`python4datascience:data-processing/serialisation-formats/csv/index`- un
  :doc:`pyviz:vega/index`-Dateien.
* :doc:`extensions` können jeden Teil von JupyterLab anpassen oder verbessern.

JupyterLab verwendet aktuell das gleiche Notebook-Dokumentenformat wie das
klassische :doc:`Jupyter Notebook <../notebook/index>`. Erst Notebook 7 wird das
klassische Jupyter Notebook-Format ersetzen.
